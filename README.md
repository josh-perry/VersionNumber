# VersionNumber

Just increments a version number in a file.

## Usage
```
> VersionNumber.exe bug
Current version: 0.0.10.4
Updated version: 0.0.10.5
```

```
> VersionNumber.exe feature
Current version: 0.0.10.5
Updated version: 0.0.11.0
```
